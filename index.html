CVDB - Cloud Vehicle Database CVDB is an product that provides organizations and individuals easy access to information about their vehicles through an API. The information provided is gathered from government vehicle registers. There two core functions of the API are: () Search vehicle(s) and 2) Fleet monitoring Search vehicles The search vehicle part of the API provides the following searches: - Search vehicle by Registration number (license plate) - Search vehicle by VIN (chassis number) - Search vehicles by Business registration number Example of information retrieved form a search: - General vehicle information (Registration number, VIN, Make, Model, Colour, Year etc.) - Vehicle status (in traffic, taken off-road etc.) - Inspection information - Technical information - Environment specifications Fleet monitoring The fleet monitoring part of the API provides functionality that monitors events for one or more vehicles. The monitoring can be setup for one individual vehicle or for all vehicles conneted to a Business registation number. Events monitored by CVDB: - Driving ban (all vehicles with driving ban, vehicles with driving ban since last report, vehicles with driving ban removed since last report) - Inspection (vehicles in need of inspection, vehicles with approved inspections since last report, vehicles with non-approved inspections since last week) - Acquired vehicles (new vehicles added to a Business registration number since last report) - Removed vehicles (vehicles removed from Business registration number since last report) - Vehicles taken off-road (all vehicle taken off-road, vehicles taken off-road since last report) - Vehicles in traffic (vehicles taken in traffic since last report)
